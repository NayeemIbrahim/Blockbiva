# Blockbiva Documentation

Thank you for purchasing Blockbiva! Fast, Flexible & Gutenberg First. This documentation will guide you through the installation and setup process.

## 1. Installation

### via WordPress Admin
1. Go to **Appearance > Themes > Add New**.
2. Click **Upload Theme**.
3. Select the `premium-theme.zip` file and click **Install Now**.
4. Activate the theme.

### via FTP
1. Unzip `premium-theme.zip`.
2. Upload the `premium-theme` folder to `/wp-content/themes/`.
3. Activate via WordPress Admin.

## 2. One Click Demo Import
1. Install and activate the **One Click Demo Import** plugin.
2. Go to **Appearance > Import Demo Data**.
3. Click "Import" on the Business Demo.
4. Wait for the process to complete.

## 3. Theme Options
Go to **Appearance > Customize** to access theme settings:
-   **Header**: Adjust layout, transparency, and logo size.
-   **Colors**: Set global primary/secondary colors.
-   **Typography**: Choose a font pairing (Modern, Classic, etc.).
-   **Performance**: Toggle built-in optimizations.

## 4. Custom Blocks
This theme includes 4 custom blocks:
-   **Blockbiva Hero**: Full-width hero sections.
-   **Call to Action**: High-conversion boxes.
-   **Pricing Table**: Modern price cards.
-   **Feature Grid**: Icon/text grids.
Search for them in the Block Editor!
