# Blockbiva Pro Documentation

Welcome to the documentation for **Blockbiva Pro**. This guide will help you set up and customize your site efficiently.

## 1. Getting Started
- **System Requirements**: PHP 7.4+, WordPress 6.4+
- **Installation**: Upload via WordPress Dashboard > Appearance > Themes.

## 2. Full Site Editing (FSE)
Premium Theme Pro is a block-based theme. You can customize every part of your site via **Appearance > Editor**.
- **Templates**: Modify Single Posts, Pages, Archives, and 404 pages visually.
- **Template Parts**: Customize the Header and Footer globally.

## 3. Custom Blocks
We include several premium blocks to enhance your content:
- **Hero Block**: Impactful introduction for your pages.
- **CTA Block**: Highly customizable Call to Action buttons.
- **Pricing Table**: Clear pricing comparison grids.
- **Features Grid**: Showcase your services or product features.

## 4. Performance & SEO
The theme is pre-optimized for speed:
- No jQuery dependency.
- Native WP Lazy Loading.
- Integrated JSON-LD Schema.

## 5. WooCommerce
Fully compatible with WooCommerce. Custom styles are automatically loaded if the plugin is active.
